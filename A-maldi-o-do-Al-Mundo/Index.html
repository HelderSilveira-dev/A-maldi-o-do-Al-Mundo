A Maldição do Alô Mundo!

Reza a lenda que que o programador que não fizer no seu primeiro código com as instruções que escreva na tela o simples Hello World ou Olá Mundo..
Não conseguirá e entender ou mesmo não será um bom profissional na área da programação…. Na minha opinião foi somente uma LendaUrbana
criada no mundo da informática por quem não tem o que fazer…. Mas por via das duvidadas…. Melhor não duvidar...

A origem deste termo se origina da linguagem C.
int main() {
printf("Hello World!\n");
return 0;
}

Este código foi escrito em 1972 por Brian Kernighan em seu livro The C Programming Language.
Desde daí o termo viralizou dentre os programadores, tanto que ainda hoje é utilizado inclusive por instituições profissionais de ensino.
